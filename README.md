# LMW Travel — Hero & Header Build

A luxury, photography-first travel agency website. This build covers the site-wide header and homepage hero section with full mobile optimization.

---

## Fonts

- **Display:** Cormorant Garamond
- **UI / Body:** Jost

Loaded via Google Fonts CDN (`@import` in `css/typography.css`).

---

## Color Palette

- Primary Warm Stone: `#726959`
- Cream Gold: `#f6e9cf`
- Deep Ink: `#1a1713`

---

## Placeholder Images

All hero and tile images are sourced from Unsplash. Replace with client photography by updating the `background-image` URLs in `css/hero.css`.

---

## Breakpoints

- Mobile: default
- Tablet: `min-width: 768px`
- Desktop: `min-width: 1024px`
- Wide: `min-width: 1440px`

---

## Notes

Open `index.html` in a browser. No build tools required.
